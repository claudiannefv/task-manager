# Task Manager API

API Node.js/Express separada do frontend.

## Como rodar

1. Instale dependências
```bash
npm install
```
2. Ambiente de desenvolvimento
```bash
npm run dev
```
3. Produção
```bash
npm start
```
4. Endereços
- API: `http://localhost:4000`
- Swagger: `http://localhost:4000/api-docs`

## Porta
- Padrão: 4000 (altere via `PORT`).
